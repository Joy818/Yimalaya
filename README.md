# Yimalaya APP

这是一个基于 Ximalaya SDK 开发的客户端，用于学习 Android App 开发流程


实现功能：

* 专辑推荐
* 专辑订阅
* 播放历史
* 专辑搜索
* 热词搜索
* 搜索关键词推荐
